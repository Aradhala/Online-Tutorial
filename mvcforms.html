MVC – Forms

MVC Form Extension

  • FormExtension class represents support for HTML in an application
  • Namespace : System.Web.Mvc.Html.FormExtensions
  • FormExtension Methods
    • FormExtensions.BeginForm Method (HtmlHelper):Writes an opening
      tag to the response. The form uses the POST method, and the request is processed by the action method for the view. The BeginForm method renders a form that will be handled by a controller action method. You can use this method in a using block. In that case, the method renders the closing
      tag at the end of the using block.
    • FormExtensions.BeginForm Method (HtmlHelper, Object): Writes an opening
      tag to the response and includes the route values in the action attribute. The form uses the POST method, and the request is processed by the action method for the view.
    • FormExtensions.BeginForm Method (HtmlHelper, RouteValueDictionary): Writes an opening <form> tag to the response and includes the route values from the route value dictionary in the action attribute. The form uses the POST method, and the request is processed by the action method for the view.
    • FormExtensions.BeginForm Method (HtmlHelper, String, String): Writes an opening tag to the response and sets the action tag to the specified controller and action. The form uses the POST method.
    • FormExtensions.BeginForm Method (HtmlHelper, String, String, FormMethod): Writes an opening tag to the response and sets the action tag to the specified controller and action. The form uses the specified HTTP method.
    • FormExtensions.BeginForm Method (HtmlHelper, String, String, FormMethod, IDictionary): Writes an opening tag to the response and sets the action tag to the specified controller and action. The form uses the specified HTTP method and includes the HTML attributes from a dictionary.
    • FormExtensions.BeginForm Method (HtmlHelper, String, String, FormMethod, Object): Writes an opening tag to the response and sets the action tag to the specified controller and action. The form uses the specified HTTP method and includes the HTML attributes. The htmlAttributes parameter consists of an object that contains name/value pairs.     • FormExtensions.BeginForm Method (HtmlHelper, String, String, Object): Writes an opening tag to the response, and sets the action tag to the specified controller, action, and route values. The form uses the POST method.
    • FormExtensions.BeginForm Method (HtmlHelper, String, String, Object, FormMethod): Writes an opening tag to the response and sets the action tag to the specified controller, action, and route values. The form uses the specified HTTP method.
    • FormExtensions.BeginForm Method (HtmlHelper, String, String, Object, FormMethod, Object): Writes an opening tag to the response and sets the action tag to the specified controller, action, and route values. The form uses the specified HTTP method and includes the HTML attributes. The htmlAttributes parameter consists of an object that contains name/value pairs. The attributes that are specified in the name/value pairs depend on the HTML element that is being rendered. For example, for a form element, you might provide the following anonymous object:
    • FormExtensions.BeginForm Method (HtmlHelper, String, String, RouteValueDictionary): Writes an opening tag to the response, and sets the action tag to the specified controller, action, and route values from the route value dictionary. The form uses the POST method.
    • FormExtensions.BeginForm Method (HtmlHelper, String, String, RouteValueDictionary, FormMethod): Writes an opening tag to the response, and sets the action tag to the specified controller, action, and route values from the route value dictionary. The form uses the specified HTTP method.
    • FormExtensions.BeginForm Method (HtmlHelper, String, String, RouteValueDictionary, FormMethod, IDictionary): Writes an opening tag to the response, and sets the action tag to the specified controller, action, and route values from the route value dictionary. The form uses the specified HTTP method, and includes the HTML attributes from the dictionary.
    • We can add attributes to a form generated from Html.BeginForm() in MVC by passing them as htmlAttributes parameters.
      • Html.BeginForm( action, controller, FormMethod, htmlAttributes )
      • Html.BeginForm( "Login", "Home", FormMethod.Post, new { id=”formname” })
      • Html.BeginForm( null, null, FormMethod.Post, new { id=”formname” })
